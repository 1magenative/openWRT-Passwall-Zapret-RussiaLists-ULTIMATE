# 🚀 OpenWRT: Passwall + Zapret + Russia Lists (ULTIMATE)

Ультимативное и полностью автоматизированное решение для обхода блокировок на роутерах с OpenWrt. 

**Как это работает после установки:**
* 🔴 **YouTube** работает мгновенно и без VPN (через систему обхода DPI — Zapret). Работает на ПК, iOS, Android и Smart TV.
* 🇷🇺 **Российские сайты** (Госуслуги, Банки, Кинопоиск) открываются напрямую через вашего провайдера (без VPN).
* 🌍 **Заблокированные сайты** (Instagram, Twitter и т.д.) открываются через ваш личный VPN (Passwall).

Вам больше не нужно вручную переключать VPN! Скрипт создаст удобную панель управления прямо в интерфейсе вашего роутера.

## ⚠️ ШАГ 0: Подготовка (Что вам понадобится)

Этот скрипт **настраивает** систему, но не устанавливает сами базовые программы (так как файлы установки отличаются для разных моделей роутеров). Перед запуском нашей "магии" у вас должно быть подготовлено:

1. Роутер на базе **OpenWrt**.
2. Программа для подключения к роутеру по SSH (например, [Termius](https://termius.com/) для телефона/ПК или PuTTY).
3. Рабочий и настроенный **Passwall** с вашим VPN-сервером.

*(💡 Совет для новичков: Установить Passwall на чистый OpenWrt бывает сложно. Проще всего скачать и прошить на роутер готовую сборку OpenWrt, где Passwall уже встроен, например от ImmortalWrt или кастомные сборки с форума 4PDA для вашей модели).*

## 📥 ШАГ 1: Правильная установка Zapret

Наш умный скрипт ожидает, что программа Zapret лежит строго в папке `/opt/zapret`. Давайте установим её туда.

1. Откройте терминал (Termius/PuTTY) и подключитесь к роутеру по SSH.
2. Установите нужные пакеты для скачивания файлов, выполнив команду:
   ```bash
   opkg update && opkg install git-http curl
Скачайте официальный Zapret напрямую в нужную папку /opt/zapret:

Bash
git clone --depth=1 [https://github.com/bolvan/zapret.git](https://github.com/bolvan/zapret.git) /opt/zapret
Перейдите в скачанную папку:

Bash
cd /opt/zapret
Запустите базовую распаковку бинарных файлов (скрипт сам подберет файлы под ваш процессор):

Bash
./install_bin.sh
Всё! Zapret установлен в правильное место. Настраивать его файл config вручную НЕ НУЖНО — наш скрипт сделает это за вас.

🪄 ШАГ 2: Установка скрипта ULTIMATE (В один клик)
Теперь, когда у вас есть Passwall и Zapret, настало время всё это связать в идеальную бесшовную систему.

Прямо в терминале роутера выполните одну команду (скопируйте и вставьте целиком):

Bash
wget -O install.sh [https://raw.githubusercontent.com/1magenative/openWRT-Passwall-Zapret-RussiaLists-ULTIMATE/main/install.sh](https://raw.githubusercontent.com/1magenative/openWRT-Passwall-Zapret-RussiaLists-ULTIMATE/main/install.sh) && sh install.sh
Что произойдет автоматически:

Скрипт пропишет в Zapret мощнейшую стратегию обхода ТСПУ (пробивает даже капризные Android и iOS приложения).

Загрузит свежие списки всех мобильных и служебных доменов YouTube.

Создаст в Passwall правило RUSSIA (для обхода VPN).

Заблокирует утечки IPv6, которые ломают загрузку видео на телефонах.

Создаст красивую Web-панель управления в меню роутера.

⚙️ ШАГ 3: Пост-настройка гео-маршрутов (Критически важно!)
Чтобы роутер понимал, какие именно сайты принадлежат России и пускал их без VPN, нужно подгрузить словари доменов (GeoSite / GeoIP).

Зайдите в веб-интерфейс вашего роутера через браузер (обычно 192.168.1.1).

Перейдите в меню: Services -> Passwall -> Rule Manage (Управление правилами).

В выпадающих списках GeoIP Update URL и Geosite Update URL выберите Loyalsoldier.

Убедитесь, что стоят галочки напротив geoip, geosite, chnroute и chnlist.

Нажмите кнопку Manually update (Обновить вручную).

Дождитесь зеленой надписи об успешном обновлении.

🎛️ Как пользоваться умной панелью?
После установки в меню вашего роутера появится новый раздел:
Services -> Passwall -> Zapret Settings

Это ваша главная панель управления Ютубом.

Нажмите «Остановить» ➔ Роутер выключит Zapret и автоматически перекинет все домены YouTube в ваш VPN (в список Proxy). Полезно, если ТСПУ провайдера временно "озверел" и Zapret перестал справляться.

Нажмите «Запустить» ➔ Роутер вернет домены YouTube напрямую к провайдеру и включит маскировку Zapret (максимальная скорость без траты трафика VPN).

Также в этой панели вы можете легко редактировать стратегию обхода (поле NFQWS_OPT), просто вставив текст и нажав "Сохранить". Роутер сам расставит кавычки и перезагрузит сервисы!

❓ Часто задаваемые вопросы (FAQ)
В: У меня перестали грузиться некоторые сайты после установки!
О: Зайдите в Services -> Passwall -> Access Control. Если там есть правило, блокирующее QUIC (UDP 443), удалите его. Наш скрипт уже блокирует QUIC точечно только для YouTube средствами самого Запрета, не ломая остальной интернет.

В: YouTube работает в браузере, но бесконечно грузится в мобильном приложении Android/iOS.
О: Мобильные приложения кэшируют маршруты. Сделайте следующее:

Выключите и включите Wi-Fi на телефоне.

Закройте приложение YouTube (смахните из недавних).

(Для Android): Зайдите в Настройки телефона -> Приложения -> YouTube -> Очистить кэш.

Убедитесь, что в телефоне отключен "Частный DNS" (Private DNS) / "Ограничение трекинга IP" (на iOS).
