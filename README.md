# OpenWRT: Passwall + Zapret + Russia Lists (ULTIMATE) 🚀

Автоматизированный скрипт для роутеров OpenWrt, который элегантно решает проблему блокировок: **YouTube работает быстро и без VPN (через обход DPI Zapret)**, все **Российские сайты идут напрямую (без VPN)**, а заблокированные ресурсы — через ваш VPN (Passwall).

Протестировано и идеально работает на: PC (Windows), iOS (iPhone/iPad) и Android (включая официальное мобильное приложение YouTube).

## ✨ Что делает этот скрипт?

Это скрипт формата "Всё-в-одном", который **не затирает ваши текущие настройки серверов**, а аккуратно добавляет нужную логику в систему через `uci`:

1. Внедряет ультимативную многоуровневую стратегию в **Zapret** (пробивает ТСПУ для TCP и жестко обрубает QUIC для мобильных приложений).
2. Создает списки всех служебных доменов Google/YouTube.
3. Настраивает маршрутизацию в **Passwall**:
   * Создает правило `RUSSIA` (весь трафик РФ идет напрямую провайдеру).
   * Автоматически направляет трафик YouTube в обход VPN, отдавая его на растерзание Zapret.
4. Отключает утечки IPv6 (AAAA записи), из-за которых тупит техника Apple и Android.
5. Устанавливает **Кастомную панель управления в LuCI** для переключения режимов в один клик.

---

## 🛠️ Требования (Перед установкой)

Так как архитектуры процессоров роутеров разные, скрипт **не устанавливает** сами пакеты. У вас уже должна быть рабочая база:
1. Установленный **OpenWrt**.
2. Установленный и работающий **Passwall** (с добавленным и рабочим VPN-узлом).
3. Установленный **Zapret** (базовый пакет).

## 📥 Установка в 1 клик

Подключитесь к вашему роутеру по SSH (например, через Termius или PuTTY) и выполните одну команду:

```bash
wget -O install.sh [https://raw.githubusercontent.com/1magenative/openWRT-Passwall-Zapret-RussiaLists-ULTIMATE/main/install.sh](https://raw.githubusercontent.com/1magenative/openWRT-Passwall-Zapret-RussiaLists-ULTIMATE/main/install.sh) && sh install.sh
