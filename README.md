## 🛠 Шаг 0: Прошивка и база
Если вы только что прошили чистый OpenWrt:
1. Подключитесь к роутеру (обычно по адресу `192.168.1.1`).
2. Установите пароль администратора в меню **System -> Administration**.
3. Убедитесь, что у роутера есть интернет (вставьте кабель провайдера в WAN порт).

---

## 💻 Шаг 1: Доступ к роутеру (SSH)
Для установки программ нам понадобится терминал.
* **Windows:** Используйте [PuTTY](https://www.putty.org/) или [Termius](https://termius.com/).
* **macOS/Linux:** Откройте стандартный терминал.

Команда для подключения:
```bash
ssh root@192.168.1.1
```
*(Введите пароль, который создали в Шаге 0)*

## 📦 Шаг 2: Установка Passwall (Основа)
Passwall — это основной шлюз. Его нет в стандартных репозиториях OpenWrt.

**Вариант А (Простой):** Прошейте сборку (например, от **ImmortalWrt** или **4PDA**), где Passwall уже встроен.
**Вариант Б (Ручной):** 
1. Узнайте архитектуру вашего процессора: `opkg print-architecture`.
2. Скачайте соответствующие пакеты `.ipk` (xray-core, luci-app-passwall и др.) из [официального репозитория](https://github.com/xiaorouji/openwrt-passwall/releases).
3. Установите их командой `opkg install <название_файла>.ipk`.

---

## 📥 Шаг 3: Установка Zapret (Движок)
Zapret обходит блокировки YouTube, не используя трафик вашего VPN. Наш скрипт ожидает, что он установлен в `/opt/zapret`.

Выполните команды по очереди:
```bash
# Обновляем пакеты и ставим зависимости
opkg update && opkg install git-http curl

# Скачиваем Zapret
git clone --depth=1 https://github.com/bolvan/zapret.git /opt/zapret

# Устанавливаем бинарные файлы (скрипт сам определит ваш процессор)
cd /opt/zapret && ./install_bin.sh
```

## 🪄 Шаг 4: Магия — Скрипт ULTIMATE
Теперь мы объединяем всё в одну систему. Наш скрипт настроит маршруты, создаст списки YouTube и **добавит панель управления в LuCI**.

Просто скопируйте и вставьте:
```bash
wget -O install.sh https://raw.githubusercontent.com/1magenative/openWRT-Passwall-Zapret-RussiaLists-ULTIMATE/main/install.sh && sh install.sh
```

**Что делает этот скрипт:**
1. Настраивает **NFQWS** (стратегия, которая пробивает ТСПУ на всех устройствах).
2. Создает список доменов YouTube (`googlevideo.com`, `ytimg.com` и др.).
3. Создает в Passwall правила **RUSSIA** (прямое соединение для банков/госуслуг).
4. **Критически важно:** Блокирует утечки IPv6 на уровне DNS, чтобы YouTube на iPhone/Android не тормозил.
5. Интегрирует панель управления в веб-интерфейс.

## 🗺 Шаг 5: Настройка Geo-фильтров
Чтобы роутер знал, какие сайты российские, нужно обновить базы данных.

1. Зайдите в **Services -> Passwall -> Rule Manage**.
2. В секции **Update Settings**:
   * **GeoIP Update URL:** Выберите `Loyalsoldier`.
   * **Geosite Update URL:** Выберите `Loyalsoldier`.
3. Нажмите **Save & Apply**, затем нажмите **Manually update** и дождитесь завершения.
4. **Важно:** В разделе **Basic Settings** выберите ваш рабочий VPN-сервер (Main Node) и нажмите сохранить.

## 🎛 Панель управления (Zapret Settings)
После установки перейдите в **Services -> Passwall -> Zapret Settings**.

* 🟢 **Запустить:** YouTube работает напрямую через вашего провайдера с маскировкой Zapret (максимальная скорость, 4K без лагов).
* 🔴 **Остановить:** Zapret выключается, и YouTube автоматически идет через ваш VPN. Это полезно, если провайдер заблокировал текущую стратегию обхода.
* ✍️ **NFQWS_OPT:** Вы можете менять параметры обхода "на лету" прямо здесь. Просто вставьте строку стратегии и нажмите "Сохранить".

## ❓ Решение проблем (FAQ)

**В: Все сайты работают, а YouTube — нет.**
**О:** Попробуйте очистить кэш браузера или приложения YouTube. На Android: *Настройки -> Приложения -> YouTube -> Очистить данные и кэш*. На iPhone: переустановите приложение или перезагрузите телефон.

**В: Перестали открываться российские сайты (банки/почта).**
**О:** Проверьте Шаг 5. Скорее всего, вы не обновили Geo-базы. Без них роутер не знает, что эти сайты не нужно пускать через VPN.

**В: Как понять, что Zapret работает?**
**О:** Зайдите на YouTube и проверьте "Статистику для сисадминов". Если видео грузится быстро, а ваш VPN-трафик при этом не растет — значит всё работает правильно.
